# Daybreak Dark Theme

A dark mode variant of the Daybreak theme with glowing accent colors.

## Design Philosophy

Daybreak Dark inverts the light theme while maintaining its bold personality. Deep backgrounds create the canvas for coral and indigo accents to shine with subtle glow effects.

## Color Palette

| Role | Color | Hex |
|------|-------|-----|
| Background Primary | Deep Navy | `#1a1a24` |
| Background Surface | Dark Blue | `#252530` |
| Background Alt | Navy | `#1f1f28` |
| Accent Primary | Coral | `#ff6b6b` |
| Accent Secondary | Indigo | `#6366f1` |
| Text Primary | White | `#ffffff` |

## Key Visual Elements

- **Glow shadows** — Colored shadows replace dark drop shadows
- **Subtle borders** — `rgba(255,255,255,0.05)` borders for depth
- **Gradient overlays** — Radial gradients for ambient color
- **High contrast text** — Pure white on dark for readability

## Use Cases

Best suited for:
- Users preferring dark mode interfaces
- Evening/nighttime viewing
- Reducing eye strain in low-light environments
