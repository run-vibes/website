# Daybreak Dark

**Mood:** Bold Premium

The confident geometric aesthetic of Daybreak, translated to a dark background. Bold and sophisticated rather than bold and accessible.

---

## Visual Identity

| Element | Value |
|---------|-------|
| Background | `#1a1a24` (warm charcoal) |
| Surface | `#252530` |
| Alt Background | `#1f1f28` |
| Primary Accent | `#ff6b6b` (brighter coral for dark bg) |
| Secondary Accent | `#6366f1` (brighter indigo for dark bg) |
| Headline Text | `#ffffff` |
| Body Text | `#a0a0a8` |

## Typography

- **Headlines:** Inter, 800 weight, slightly condensed
- **Body:** Inter, 400 weight
- **Accents:** Bold, coral-colored highlights

## Hero Visual: Geometric Shapes

Same SVG shapes and parallax behavior as Daybreak, adjusted for dark:

- Brighter shape fills for contrast on dark background
- Subtle glow/shadow effect added to shapes
- Shapes feel like they're floating on the dark surface
- Mouse parallax unchanged

## Interaction Style

| Element | Behavior |
|---------|----------|
| Cards | Dark surface with soft coral/indigo border glow on hover |
| Primary Button | Coral fill, white text |
| Secondary Button | Indigo outline with glow |
| Navigation | Dark with blur, coral accent on CTA |
| Section Backgrounds | Alternating dark tones |

## Scroll Animations

- Sections slide from left/right (alternating)
- Cards scale up on reveal
- Snappy, energetic easing (same as Daybreak)
