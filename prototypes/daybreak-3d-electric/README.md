# Daybreak 3D Electric

Bold and modern with electric purple and cyan.

## Color Palette

| Role | Color | Hex |
|------|-------|-----|
| Primary | Violet | `#8b5cf6` |
| Secondary | Cyan | `#06b6d4` |
| Tertiary | Near White | `#f8fafc` |

## Design Notes

This is a color variant of the Daybreak 3D theme. It shares the same:
- 3D floating shapes (cubes, wireframes, pyramids)
- Mouse-reactive tilt behavior
- Animation timing and easing
- Layout and typography

See `../daybreak-3d/README.md` for full technical details.
