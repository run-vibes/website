# Midnight Light Theme

A light mode variant of the Midnight theme, preserving the electric accents.

## Design Philosophy

This theme takes Midnight's electric cyan and hot pink accents and places them against light backgrounds. The result maintains the energetic, tech-forward feel while improving readability in bright environments.

## Color Palette

| Role | Color | Hex |
|------|-------|-----|
| Background Primary | Off-White | `#fafafa` |
| Background Surface | White | `#ffffff` |
| Accent Primary | Electric Cyan | `#00d4ff` |
| Accent Secondary | Hot Pink | `#ff2d7a` |
| Text Primary | Near Black | `#111111` |

## Key Visual Elements

- **Vibrant accents** — Same electric colors as Midnight
- **Light shadows** — Subtle depth without darkness
- **Clean cards** — White surfaces with soft shadows
- **Color pops** — Strategic use of cyan and pink

## Use Cases

Best suited for:
- Users preferring light interfaces with bold accents
- Daytime/well-lit viewing environments
- Tech brands wanting accessibility with personality
